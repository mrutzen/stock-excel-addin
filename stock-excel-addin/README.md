# Stock Data Puller — Excel Add-in

Pull live stock quotes, fundamentals, and historical OHLCV data from Yahoo Finance directly into any Excel sheet.

---

## What gets written to the sheet

| Section | Contents |
|---|---|
| **Header** | Company name, ticker, exchange, sector, industry, timestamp |
| **Current Quote** | Price, change %, volume, market cap, open, day high/low, prev close |
| **Fundamentals** | P/E, EPS, revenue, margins, ROE, ROA, debt/equity, cash flow |
| **Valuation & Dividends** | 52-wk range, moving averages, beta, shares, dividends, book value |
| **Historical Prices** | Daily OHLCV for the selected time range (up to 5 years) |

Data starts at **A1** and overwrites existing content in the active sheet.

## Quick Start

### Prerequisites

- Node.js 18+ (https://nodejs.org)
- Microsoft Excel (desktop, Microsoft 365 — not Excel Online)
- Windows or macOS

### 1. Install dependencies

```bash
cd stock-excel-addin
npm install
```

### 2. Run setup (generates icons + installs HTTPS dev certs)

```bash
node setup.js
```

> On Windows you may see a UAC prompt — accept it so the certificate can be trusted.
> On macOS you may be asked for your login password.

### 3. Start the local server

```bash
npm start
```

The server runs at `https://localhost:3000`.

### 4. Sideload the add-in in Excel

**Excel for Windows (Microsoft 365):**

1. Open Excel and go to **Insert → Get Add-ins → Manage My Add-ins → Upload My Add-in**
2. Browse to `manifest.xml` in this folder and click **Upload**
3. A **"Stock Data"** button appears in the **Home** ribbon

**Excel for Mac:**

1. Go to **Tools → Excel Add-ins → Browse…**
2. Select `manifest.xml`

> The add-in appears on the **Home** tab in the **Stock Data** group.

### 5. Use the add-in

1. Click **Stock Data** in the ribbon to open the task pane
2. Type any ticker symbol (e.g. `AAPL`, `MSFT`, `BRK-B`, `^GSPC`)
3. Choose a time range for historical data
4. Click **Pull Stock Data** — data populates from A1

## Troubleshooting

| Problem | Fix |
|---|---|
| "Ticker not found" | Double-check the symbol on finance.yahoo.com |
| Server won't start | Make sure port 3000 is free; change `PORT` in server.js if needed |
| HTTPS cert error in browser | Visit `https://localhost:3000`, accept the self-signed cert |
| Add-in won't load | Ensure the server is running before opening Excel |
| Certificate error on sideload | Re-run `node setup.js` or run `npx office-addin-dev-certs install` |

## Project structure

```
stock-excel-addin/
├── manifest.xml          ← Excel add-in manifest (sideload this)
├── server.js             ← Express server (HTTPS + Yahoo Finance proxy)
├── setup.js              ← One-time setup: icons + dev certs
├── package.json
└── src/
    ├── taskpane.html     ← Task pane UI
    ├── taskpane.css      ← Styles
    ├── taskpane.js       ← Office.js logic + data writing
    ├── commands.html     ← Required by manifest (empty command file)
    └── assets/
        ├── icon-16.png   ← Generated by setup.js
        ├── icon-32.png
        └── icon-80.png
```

## Notes

- Data is sourced from Yahoo Finance's unofficial public API — no API key required.
- The local server acts as a proxy to avoid browser CORS restrictions.
- For production use, deploy the `src/` folder to any HTTPS host and update all `localhost:3000` URLs in `manifest.xml`.
